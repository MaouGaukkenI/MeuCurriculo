Linguagens de Programação

  • Java - Conceitos Abordados: Classes, objetos, herança, polimorfismo, encapsulamento, exceções, coleções, e manipulação de arquivos.

  • JavaScript - Conceitos Abordados: Manipulação de DOM, eventos, AJAX (para requisições assíncronas), manipulação de dados em tempo real, e integração com APIs.

  • HTML e CSS - Conceitos Abordados: Criação de páginas estáticas e responsivas, manipulação de elementos HTML, posicionamento e layout com CSS, e técnicas de design como Flexbox e Grid.

  • SQL - Conceitos Abordados: Criação de tabelas, relacionamentos entre dados, consultas (SELECT), agregações (COUNT, SUM, AVG), e manipulação de dados (INSERT, UPDATE, DELETE).

  • Git - Conceitos Abordados: Commits, branches, merges, clones, e operações de recuperação de versões anteriores.

  • JSON - Conceitos Abordados: Estrutura de objetos e arrays, conversão de JSON para objetos e vice-versa (JSON.parse() e JSON.stringify()), validação de dados com JSON Schema, e uso de JSON em APIs para troca de dados entre cliente e servidor.
